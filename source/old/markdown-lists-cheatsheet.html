Markdown lists cheatsheet

Published: Sun 29 January 2017
By iaroki

In Markdown.

tags: markdown

To create unordered lists add asterisk:

* First item
* Second item
* Third item

Ordered lists can be created by addint number before item:

1. First item
2. Second item
3. Third item
